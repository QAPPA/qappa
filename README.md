# QAPPA

> Quality Agile Project Planning Application

## Basic Setup

``` bash
# install dependencies
$ yarn

# serve client
$ yarn client:dev

#serve server
$ yarn server:dev

# build client for production and launch client app
$ yarn client:build
$ yarn client:start

# transpile server files and launch it
$ yarn server:build
$ yarn server:start
```
